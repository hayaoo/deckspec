# Changelog

## 0.1.0 (2026-03-27)

Initial release.

- YAML DSL for declarative slide authoring
- Zod schema validation for pattern vars
- React SSR pattern rendering
- CLI: `init`, `validate`, `render`, `dev`, `approve`, `lock`, `patterns`
- Theme: noir-display (17 patterns)
- Deck-local pattern support with on-the-fly esbuild compilation
- Slide approval workflow (generated → approved → locked)
- Dev server with live preview dashboard
